Cascade

A collection of pacakage, plugins and strategies to introduce safetyness into Flutter development with the goal of reducing the amount of boilerplate code and the amount of bugs. The packages try to acomplish this by:

  • Navigator type safety into Modals, Dialogs, BottomSheets and pushed routes.
  • Page lifecycle management.
  • State management minimization.
  • Safe Context management.
  • Dependency injection.
  • Modularization.
  • Fronted validation.
  • Semi-Metaprogramming.

These packages are designed to be used together, but they can be used independently.
